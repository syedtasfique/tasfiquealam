About Me

With over 5 years of professional experience, I specialize in designing and developing web and mobile applications for various U.S.-based clients. I have a strong command of Agile methodologies and the full Software Development Life Cycle, from requirement analysis to deployment. My expertise lies in JavaScript and modern frontend frameworks such as React.js, React Native, Next.js, and Vue.js. I follow clean coding practices and design principles like SOLID to build scalable, maintainable applications.

I’m proficient in version control using Git and have hands-on experience with performance monitoring tools like New Relic and LogRocket, enabling efficient debugging, transaction tracing, and user session analysis. I have a solid understanding of responsive UI/UX design, and I’m skilled in HTML5, CSS3, Tailwind CSS, Bootstrap, AJAX, and SASS. My development toolkit includes Visual Studio Code, IntelliJ IDEA, Postman, and JIRA. I’m also experienced in working with RESTful APIs, handling JSON data, and building cross-platform mobile apps for both Android and iOS using React Native. I’ve developed and deployed applications across multiple platforms including Windows, Mac, Linux, Android, and iOS.
